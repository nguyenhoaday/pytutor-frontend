# PyTutor AI Frontend

Frontend (React + TypeScript + Vite) cho hệ thống luyện tập Python + AI Tutor.

## Requirements

- Node.js 18+ (khuyến nghị)

## Config

Tạo file `.env` (hoặc dùng biến môi trường hệ thống):

```env
VITE_API_URL=http://localhost:8000
```

## Run

```bash
npm install
npm run dev
```

Mặc định Vite chạy ở `http://localhost:3000`.
